# cnBeta
一款cnBeta新闻第三方客户端。   
主要功能包括：   
＊流畅的新闻阅读：最新资讯、热点资讯；   
＊新闻列表显示缩略图；   
＊新闻详情初次加载后缓存；   
＊新闻评论显示热门评论和全部评论；   
＊评论盖楼效果；   
＊支持发表评论、回复评论、支持和反对；   
＊新闻收藏.
